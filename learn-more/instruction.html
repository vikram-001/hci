Instructions

1. Read all information about Stroop test
2. Listen to pleasant music,make yourself calm and focused
3. Click on play button
4. Select no of players need to play
5. Enter your name
6. There are two way to start the test:

  • Select the square that corresponds to color that the word is For Example: Blue You need to choose Red box from the options given
  • Select the square that corresponds to what the word is For Example: Blue You need to choose Blue box from the options given

7. Timer will start as soon as button is pressed
8. After all players completing their test- Leader board will be displayed
